## 安裝gemini cli
1. 先安裝node.js
2. 安裝gemini cli ，使用指令
```
npm install -g @google/gemini-cli
```
3. 在terminal使用指令gemini就可以開啟。第一次可設定theme，也會問你要怎麼連到google？請依照自己的情形來選擇。

## 簡單的指令
1. Gemini cli說明，可參考網址：https://geminicli.com/docs/cli/commands/
2. 簡單三個指令很好用：
    - ```/``` （斜線可以帶出包裹好的命令給你選，譬如 /settings ，就進入相關設定）
    - ```@``` （你可以用此符號指定檔案，然後請ai操作，譬如分析）
    - ```!``` （驚嘆號可以使用在terminal使用的指令，譬如ls來瀏覽目錄）

## 開始寫程式：AI你畫我猜
### 申請google api key
- 到[Google AI Studio](https://aistudio.google.com/app/api-keys)去申請api key
- 要注意api key不要寫在檔案中，應該要寫在環境變數中，不然你的帳單會爆掉。即便你使用免費一年方案，但倘若key被盜，也會有次數被用盡而無法使用的問題。

### 請gemini幫忙寫功能
- 直接在gemini cli寫這樣的prompt，然後都approve他的要求。
```
我想要你幫我做一個你畫我猜的游戲。
需要使用者,也就是我來畫,而AI來猜我畫什麽。
用的模型是gemini-2.0-flash。
這是gemini的apikey=(這裏填上你的API_KEY)。
繪畫的功能都要有,像是橡皮擦,填滿,上一步,還原,清空屏幕。
畫完后按確認鍵讓AI來猜。
```
- 慢慢等他跑完，然後就啟動他寫好的檔案。助教demo是用node.js。而我的ai卻是用python寫。所以我的情況是，又要依照目錄中的requirement.txt來安裝import一些函式。所以我在目錄中，先安裝虛擬環境venv，避免造成混亂。
